Tjek siden validation for mere info om hvordan du kan teste kvaliteten af dit eget JSON-objekt

V1 - South Park

"schema" datamodel

  • name: tekst
  • nickname: tekst
  • image: url
  • occupation: tekst (student, teacher, etc.)
  • age: number
  • voicedBy: tekst
  • gender: tekst: male, female, …
  • religion: tekst: jewich, christian, scientology, ...
  • catchPhrase: tekst
  • hairColor: tekst
  • schoolGrade: tal 1-10
  • episodes: tekst, med kommasepareret liste af fx S01E01, S01E04, S04E11, .. eller undefined
  • appearances: tal
  • firstAppearance: tekst, episode id, som “S01E01”

V2 - Pokemon

"Schema" datamodel

  • name: tekst
  • description: tekst
  • ability: tekst
  • image: url
  • footprint: url (til et andet billede)
  • dexindex: tal
  • type: tekst – begrænset til: fire, ice, flying, etc …
  • subtype: tekst
  • weaknesses: tekst – en kommasepareret liste over types
  • gender: tekst: male eller female
  • weight: tal – vægt i gram
  • height: tal – højde i cm
  • generation: tal
  • spilversion: tekst
  • canEvolve: boolean
  • statsHP: tal 0-10
  • statsAttack: tal 0-10
  • statsDefence: tal 0-10
  • statsSpecialAttack: tal 0-10
  • statsSpecialDefence: tal 0-10
  • statsSpeed: 0-10
